# CLAUDE.md

## Project

Visio.ai marketing website and blog. Built from the first-party Astro blog starter template: https://github.com/withastro/astro/tree/main/examples/blog. The blog template is the base for the most popular community themes (AstroPaper, etc.).

GitHub repo: https://github.com/stuartbrameld/visio-website

## Package Manager

Use **pnpm** (not npm) for all package management.

## Commands

- `pnpm dev` — Start the dev server
- `pnpm build` — Production build
- `pnpm preview` — Preview production build

## Brand

- **Colors**: Teal/turquoise palette from visio.ai brand (#1fc0a2, #20bccc, #146770) plus blue-to-green gradient accents (#00A5DB → #71C27D) matching the logo
- **Fonts**: Montserrat (headings) + Inter (body) via Google Fonts
- **Brand & design system**: For brand guidelines, visual identity, and positioning, see https://github.com/Visio-ai/arch-design-system/tree/master/domains
- **Design direction**: Matheus (CEO) wants a more premium feel — prioritise refinement, generous whitespace, and polished details over flashy effects
